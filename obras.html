Propósito

Esta página documenta de forma transparente iniciativas, apoyos y obras sociales. El objetivo es mantener un registro ordenado, verificable y actualizado.

Sugerencia: para cada obra, registrar fecha, lugar, objetivo, actividades realizadas, evidencias (si aplica) y resultados.

Para añadir una nueva obra: copia una “card”, cambia título/fecha/lugar/estado y guarda cambios.

Cómo publicar una obra (plantilla rápida)

  1. Agregar una nueva tarjeta en esta página (obras.html).
  2. Opcional: crear una página de detalle en /obras/AAAA-MM-obra-XX.html.
  3. Si hay evidencias (fotos), súbelas a /assets/fotos/ y enlázalas.
  4. Actualizar el estado: Planificada → En ejecución → Finalizada.

Si quieres, te preparo la plantilla de “página de detalle” para cada obra (con secciones y formato estándar).
